SALIKTEŅI
(Составные слова)

💡 Что такое составные слова в латышском языке?
Составные слова образуются, объединяя по крайней мере два самостоятельных слова или их части. Есть несколько способов образования составных слов.

📋 Способы образования составных слов

Способ образования Описание Примеры
1. Оба слова полностью сохраняют свою форму Оба слова остаются без изменений saules + puķe → saulespuķe
(солнце + цветок → подсолнух)

ūdens + roze → ūdensroze
(вода + роза → водяная лилия)
2. Первое слово теряет окончание У первого слова убирается окончание saruna + biedrs → sarunbiedrs
(разговор + товарищ → собеседник)

viedais + tālrunis → viedtālrunis
(умный + телефон → смартфон)

mašīna + tulks → mašīntulks
(машина + переводчик → машинный переводчик)
3. Второе слово получает другое окончание Второе слово изменяет своё окончание bērzs + lapa → bērzlape
(берёза + лист → берёзовый лист)

brūns + acs → brūnacis / brūnace
(коричневый + глаз → кареглазый)

🎯 Подробные примеры

Способ 1: Оба слова полностью сохраняют форму

Способ 3: Второе слово получает другое окончание

  • bērzs + lapa → bērzlape (берёзовый лист)
  • brūns + acs → brūnacis / brūnace (кареглазый)
  • zaļš + acs → zaļacis / zaļace (зеленоглазый)

  • balts + acs → baltacis / baltace (белоглазый)

⚠️ Важные особенности

🔍 Особенности составных слов в латышском языке:
  • Составные слова образуются из двух или более самостоятельных слов
  • Есть три основных способа образования составных слов
  • В первом способе оба слова полностью сохраняют свою форму
  • Во втором способе первое слово теряет окончание
  • В третьем способе второе слово получает другое окончание
  • Составные слова часто обозначают новые понятия
  • Они помогают экономить слова и делают речь более точной
  • Составные слова могут быть существительными, прилагательными и другими частями речи
💡 Практический совет: При изучении составных слов обращайте внимание на то, какой способ образования используется. Запоминайте наиболее распространённые составные слова. Попробуйте сами образовывать составные слова, используя известные вам слова. Составные слова часто встречаются в современном латышском языке, особенно в технических и научных текстах.
