(Сложные времена страдательного залога)

💡 Что такое сложные времена страдательного залога в латышском языке?
Сложные времена страдательного залога образуются с помощью вспомогательного глагола "būt" (быть) в разных временах и причастия страдательного залога прошедшего времени. Они выражают завершённое действие в страдательном залоге.

📝 Образование сложных времен страдательного залога

Структура сложных времен страдательного залога:

Сложные времена страдательного залога образуются по формуле:

  • Глагол "būt" + причастие страдательного залога прошедшего времени
  • Причастие согласуется с подлежащим по роду и числу
  • Мужской род: -ts (ед.ч.), -ti (мн.ч.)
  • Женский род: -ta (ед.ч.), -tas (мн.ч.)

Примеры образования:

  • uzaicinā-t (пригласить) → uzaicinā-ts/uzaicinā-ta (приглашённый/приглашённая)
  • rakstī-t (написать) → rakstī-ts/rakstī-ta (написанный/написанная)
  • lasī-t (прочитать) → lasī-ts/lasī-ta (прочитанный/прочитанная)
  • būvē-t (построить) → būvē-ts/būvē-ta (построенный/построенная)

⚠️ Важные особенности

🔍 Особенности сложных времен страдательного залога:
  • Сложные времена страдательного залога образуются с помощью глагола "būt"
  • Глагол "būt" спрягается по временам: esmu/esi/ir (наст.), biju/biji/bija (прош.), būšu/būsi/būs (буд.)
  • Причастия страдательного залога согласуются с подлежащим по роду и числу
  • Мужской род: -ts/-ti, женский род: -ta/-tas
  • Сложные времена выражают завершённое действие в страдательном залоге
  • Цветовое выделение помогает различать мужской и женский род
  • Объект действия становится подлежащим предложения
  • Исполнитель действия может быть выражен творительным падежом
