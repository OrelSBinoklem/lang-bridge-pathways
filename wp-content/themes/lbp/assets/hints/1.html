📚 Склонение существительных (S1)

📌 Основная информация: Эта таблица показывает полную систему склонения латышских существительных в единственном и множественном числе для мужского и женского рода.

🔍 Структура таблицы

Вертикальное деление:
• Vienskaitlis - единственное число (верхняя часть)
• Daudzskaitlis - множественное число (нижняя часть)
Горизонтальное деление:
• Vīriešu dzimte - мужской род (левая сторона)
• Sieviešu dzimte - женский род (правая сторона)
Деление по цифрам:
Это называется "Деклинациями"
3 и 6 деклинации являются исключениями и очень редкие

📖 Падежи и вопросы

Падеж Название Вопрос Слово помощник
Nominatīvs Именительный Kas? (Кто?) Kas ir? (Кто есть?)
Ģenitīvs Родительный Kā? (Кого?) Kā nav? (Кого нет?)
Datīvs Дательный Kam? (Кому?) Kam es dodu? (Кому я даю?)
Akuzatīvs Винительный Ko? (Кого? Что?) Ko es redzu? (Кого я вижу?)
Instrumentālis Творительный Ar ko? (С кем?) Ar ko es runāju? (С кем я говорю?)
Ar ko es lepojos? (Кем я горжусь?)
Par ko? Предложный Par ko? (О ком? О чём?) Par ko es domāju? (О чём я думаю?)
Lokatīvs Местный Kur? (Где?) Kur es esmu? (Где я нахожусь?)
Vokatīvs Звательный — —
💡 Важная особенность латышского: Падежи творительный и предложный в латышском языке склоняются как винительный падеж (Akuzatīvs)! Также в латышском нет предложного падежа (но можно думать что есть)!

👨 Мужской род - примеры

tēvs (отец):
tēvs → tēva → tēvam → tēvu → tēvā → tēv!
отец → отца → отцу → отца → об отце → отец! (обращение)
brālis (брат):
brālis → brāļa → brālim → brāli → brālī → brāli!
брат → брата → брату → брата → о брате → брат! (обращение)
* Обратите внимание на чередование согласных (ļ → l)
lietus (дождь):
lietus → lietus → lietum → lietu → lietū → liet!
дождь → дождя → дождю → дождь → о дожде → дождь! (обращение)

👩 Женский род - примеры

māsa (сестра):
māsa → māsas → māsai → māsu → māsā → māsa!
сестра → сестры → сестре → сестру → о сестре → сестра! (обращение)
upe (река):
upe → upes → upei → upi → upē → up!
река → реки → реке → реку → о реке → река! (обращение)
pils (замок):
pils → pils → pilij → pili → pilī → pil!
замок → замка → замку → замок → о замке → замок! (обращение)

🔢 Множественное число

Мужской род:
tēvi → tēvu → tēviem → tēvus → tēvos
отцы → отцов → отцам → отцов → об отцах
Женский род:
māsas → māsu → māsām → māsas → māsās
сестры → сестёр → сёстрам → сестёр → о сёстрах

📢 Звательный падеж (Vokatīvs)

Примеры обращений:
Tēv! (Отец!)
Отец! (обращение к отцу)
Māsa! (Сестра!)
Сестра! (обращение к сестре)
Brāli! (Брат!)
Брат! (обращение к брату)

💡 Ключевые особенности

  • Чередование согласных - отмечено звёздочкой (*) в таблице
  • Окончания падежей зависят от рода и склонения
  • Вопросы помогают определить нужный падеж
  • Множественное число имеет свои особенности склонения
⚠️ Важно запомнить: В латышском языке род существительного влияет на его склонение, поэтому важно правильно определить род слова!
📝 Практический совет: Начинайте изучение с основных форм (именительный и родительный падежи), а затем постепенно добавляйте остальные падежи.
