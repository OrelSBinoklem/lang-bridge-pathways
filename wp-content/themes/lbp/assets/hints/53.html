PAVĒLES IZTEIKSME
(Повелительное наклонение)

💡 Что такое повелительное наклонение (pavēles izteiksme) в латышском языке?
Повелительное наклонение используется для выражения приказа, просьбы, совета или пожелания. Оно имеет разные формы в зависимости от лица, к которому обращается говорящий.

📝 Образование повелительного наклонения

Правила образования повелительного наклонения:

  • 1-е лицо единственного числа (es): повелительное наклонение НЕ образуется
  • 2-е лицо единственного числа (tu): форма совпадает с настоящим временем + восклицательный знак
  • 3-е лицо (viņš, viņa, viņi, viņas): lai + форма настоящего времени + восклицательный знак
  • 1-е лицо множественного числа (mēs): основа + -sim/-simies + восклицательный знак
  • 2-е лицо множественного числа (jūs): основа + -iet/-ieties + восклицательный знак

Примеры образования:

  • nākt (приходить):
    • tu: nāc! (приходи!)
    • viņš: lai nāk! (пусть приходит!)
    • mēs: nāksim! (пойдём!)
    • jūs: nāciet! (приходите!)
  • celties (вставать):
    • tu: celies! (вставай!)
    • viņš: lai ceļas! (пусть встаёт!)
    • mēs: celsimies! (встанем!)
    • jūs: celieties! (вставайте!)

⚠️ Важные особенности

🔍 Особенности повелительного наклонения:
  • Повелительное наклонение используется для приказов, просьб, советов
  • 1-е лицо единственного числа (es) НЕ имеет повелительного наклонения
  • 2-е лицо единственного числа: форма совпадает с настоящим временем
  • 3-е лицо: используется частица "lai" + форма настоящего времени
  • 1-е лицо множественного числа: основа + -sim/-simies
  • 2-е лицо множественного числа: основа + -iet/-ieties
  • Цветовое выделение: синий для обычных глаголов, розовый для возвратных, зелёный для восклицательных знаков
  • Все формы заканчиваются восклицательным знаком
💡 Практический совет: Запомните правила образования повелительного наклонения для разных лиц. Обратите внимание на использование частицы "lai" для 3-го лица. Повелительное наклонение всегда заканчивается восклицательным знаком.
