AR PRIEDĒKĻIEM ATVASINĀTI DARBĪBAS VĀRDI
(Глаголы, образованные с помощью приставок)

💡 Что такое глаголы с приставками в латышском языке?
Приставка изменяет значение глагола: глагол с приставкой не означает то же самое, что глагол без приставки, хотя основное значение обычно похоже. Глагол с приставкой часто указывает на завершённое действие с результатом.

📋 Основные приставки в латышском языке

Приставки глаголов:

Глагольные приставки в латышском языке:

  • aiz- (за-, у-)
  • ap- (около-, об-)
  • at- (от-, обратно)
  • ie- (в-, в-)
  • iz- (из-, вы-)
  • no- (с-, от-)
  • pa- (под-, немного)
  • pār- (пере-, через)
  • pie- (при-, к-)
  • sa- (с-, со-)
  • uz- (на-, вверх)

Важные особенности:

  • Не каждая приставка может соединяться с любым глаголом
  • Многие глаголы используются с несколькими приставками
  • Глагол с приставкой часто указывает на завершённое действие с результатом
  • Значение завершённости может быть у всех приставок

🎯 Примеры глаголов с приставками

rakstīt (писать):

  • rakstīja (писал) → uzrakstīja (написал)
  • rakstīja (писал) → sarakstīja (составил письменно)
  • rakstīja (писал) → pierakstīja (записал)
  • rakstīja (писал) → norakstīja (списал)
  • rakstīja (писал) → aizrakstīja (записал)
  • rakstīja (писал) → atrakstīja (ответил письмом)
  • rakstīja (писал) → ierakstīja (вписал)
  • rakstīja (писал) → pārrakstīja (переписал)

sūtīt (посылать):

  • sūtīja (посылал) → aizsūtīja (отослал)
  • sūtīja (посылал) → izsūtīja (разослал)
  • sūtīja (посылал) → nosūtīja (отправил)
  • sūtīja (посылал) → pasūtīja (заказал)
  • sūtīja (посылал) → pārsūtīja (переслал)

ēst (есть):

  • ēda (ел) → paēda (поел немного)
  • ēda (ел) → apēda (съел всё)
  • ēda (ел) → izēda (съел всё)

stāstīt (рассказывать):

  • stāstīja (рассказывал) → pastāstīja (рассказал)
  • stāstīja (рассказывал) → izstāstīja (выложил всё)
  • stāstīja (рассказывал) → pārstāstīja (пересказал)

darīt (делать):

  • darīja (делал) → padarīja (сделал)
  • darīja (делал) → izdarīja (выполнил)

📚 Возвратные глаголы с приставками

klausīties (слушать):

  • klausījās (слушал) → noklausījās (выслушал)

skatīties (смотреть):

  • skatījās (смотрел) → apskatījās (осмотрелся)
  • skatījās (смотрел) → noskatījās (посмотрел)

mazgāties (мыться):

  • mazgājās (мылся) → nomazgājās (вымылся)

peldēties (плавать):

  • peldējās (плавал) → izpeldējās (выплавался)
  • peldējās (плавал) → nopeldējās (наплавался)

📝 Дополнительные примеры

Другие глаголы с приставками:

  • lasīt (читать) → izlasīja (прочитал), pārlasiīja (перечитал)
  • runāt (говорить) → izrunāja (выговорил), pārrunāja (переговорил)
  • dziedāt (петь) → izdziedāja (выпел), pārdziedāja (перепел)
  • strādāt (работать) → nostrādāja (отработал), pārstrādāja (переработал)
  • mācīt (учить) → izmācīja (выучил), pamācīja (научил)
  • redzēt (видеть) → pamanīja (заметил), ieredzēja (увидел)
  • dzirdēt (слышать) → nodzirdēja (услышал), iedzirdēja (услышал)
  • iet (идти) → aizgāja (ушёл), iegāja (вошёл), izgāja (вышел)

⚠️ Важные особенности

🔍 Особенности глаголов с приставками:
  • Приставка изменяет значение глагола
  • Глагол с приставкой часто указывает на завершённое действие
  • Приставка добавляет результат к действию
  • Не каждая приставка может соединяться с любым глаголом
  • Многие глаголы используются с несколькими приставками
  • Значение завершённости может быть у всех приставок
  • Приставки могут кардинально изменить значение глагола
  • Возвратные глаголы также могут иметь приставки
💡 Практический совет: Изучайте глаголы с приставками систематически. Обратите внимание на то, как приставка изменяет значение глагола. Запоминайте наиболее распространённые сочетания приставок с глаголами. Приставки часто указывают на направление, завершённость или интенсивность действия.

📝 Практические упражнения

Образуйте глаголы с приставками:

  • rakstīt + uz- → uzrakstīt (написать)
  • sūtīt + aiz- → aizsūtīt (отослать)
  • ēst + pa- → paēst (поесть немного)
  • stāstīt + iz- → izstāstīt (выложить всё)
  • darīt + iz- → izdarīt (выполнить)
  • klausīties + no- → noklausīties (выслушать)
  • skatīties + ap- → apskatīties (осмотреться)
  • peldēties + iz- → izpeldēties (выплаваться)

Определите значение глаголов с приставками:

  • uzrakstīt → написать (завершить письмо)
  • izsūtīt → разослать (отправить многим)
  • apēst → съесть всё (полностью)
  • pārstāstīt → пересказать (повторить рассказ)
  • izdarīt → выполнить (завершить дело)
  • noklausīties → выслушать (до конца)
  • apskatīties → осмотреться (вокруг)
  • nomazgāties → вымыться (полностью)

Переведите на латышский:

  • написать письмо → uzrakstīt vēstuli
  • отослать посылку → aizsūtīt sūtījumu
  • поесть немного → paēst nedaudz
  • рассказать историю → pastāstīt stāstu
  • выполнить работу → izdarīt darbu
  • выслушать музыку → noklausīties mūziku
  • осмотреться вокруг → apskatīties apkārt
  • выплаваться в море → izpeldēties jūrā
