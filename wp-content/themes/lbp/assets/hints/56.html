VĀRDA SASTĀVS
(Структура слова)

💡 Что такое структура слова в латышском языке?
В латышском языке слова состоят из корня и окончания. Они также могут иметь приставку и суффикс. Понимание структуры слова помогает лучше понимать язык и образовывать новые слова.

📋 Компоненты слова (Vārda sastāvdaļas)

Vārda sastāvdaļas
(Компоненты слова)		 Raksturojums
(Описание)		 Piemērs
(Пример)
Sakne
(Корень)		 Часть слова, которая является общей для семьи родственных слов. Эта общность выражается в лексическом значении слова. Корень является обязательным компонентом слова. Слова могут иметь несколько корней. valde, valdīt, pārvaldīt, valdnieks, valdība, pārvalde, pārvaldnieks
Galotne
(Окончание)		 Часть слова, которая находится в конце слова и изменяется при склонении слова. saulespuķes
kok-s, kok-a, kok-am, kok-u, kok-ā
Priedēklis
(Приставка)		 Часть слова, которая находится перед корнем. Слова могут иметь несколько приставок. pa-saule, aiz-krāsne, ie-pa-tikties, jā-no-raksta
Piedēklis
(Суффикс)		 Часть слова, которая находится между корнем и окончанием. Слова могут иметь несколько суффиксов. dej-o-t, skol-niek-s, run-ā-j-a
Celms
(Основа)		 Все слово без окончания. strādniek-s, pasaul-e
Izskaņa
(Финальная часть)		 Часть слова, которая включает последний суффикс вместе с окончанием. strād-nieks, soļo-tājs, aizzīmog-ot

📝 Типы слов по структуре

Слова могут быть:

  • Pirmatnīgi (Первоначальные) - образуются корнем и окончанием
  • Atvasināti (Производные) - кроме корня и окончания имеют также приставку и/или суффикс

📚 Дополнительные примеры структуры слов

Слова с приставками:

  • pa-saule (мир) = pa- + saule
  • aiz-krāsne (камин) = aiz- + krāsne
  • ie-pa-tikties (понравиться) = ie-pa- + tikties
  • jā-no-raksta (переписать) = jā-no- + raksta
  • uz-būvēt (построить) = uz- + būvēt
  • at-griezt (вернуть) = at- + griezt

Слова с суффиксами:

  • dej-o-t (танцевать) = dej- + -o- + -t
  • skol-niek-s (ученик) = skol- + -niek- + -s
  • run-ā-j-a (говорящая) = run- + -ā-j- + -a
  • strād-ā-tājs (работник) = strād- + -ā-tāj- + -s
  • māc-ī-tājs (учитель) = māc- + -ī-tāj- + -s
  • rakst-ī-tājs (писатель) = rakst- + -ī-tāj- + -s

Слова с несколькими компонентами:

  • pār-vald-niek-s (управляющий) = pār- + vald- + -niek- + -s
  • ne-saprot-āms (непонятный) = ne- + saprot- + -ām- + -s
  • uz-būv-oj-am (строим) = uz- + būv- + -oj- + -am
  • at-griez-ī-t (повернуть) = at- + griez- + -ī- + -t

⚠️ Важные особенности

🔍 Особенности структуры слова в латышском языке:
  • Корень - обязательная часть слова, общая для родственных слов
  • Окончание - изменяемая часть, которая находится в конце слова
  • Приставка - находится перед корнем, может быть несколько
  • Суффикс - находится между корнем и окончанием, может быть несколько
  • Основа - всё слово без окончания
  • Финальная часть - последний суффикс + окончание
  • Цветовое выделение: оранжевый для приставок, синий для корней, розовый для суффиксов, зелёный для окончаний
💡 Практический совет: Понимание структуры слова помогает лучше понимать значение слов и образовывать новые. Обратите внимание на корни - они часто повторяются в родственных словах. Приставки и суффиксы могут кардинально изменить значение слова.

📝 Практические упражнения

Разберите структуру слов:

  • strādnieks → strād- (корень) + -niek- (суффикс) + -s (окончание)
  • pārvaldnieks → pār- (приставка) + vald- (корень) + -niek- (суффикс) + -s (окончание)
  • uzbūvēt → uz- (приставка) + būv- (корень) + -ē- (суффикс) + -t (окончание)
  • nesaprotāms → ne- (приставка) + saprot- (корень) + -ām- (суффикс) + -s (окончание)
  • mācītājs → māc- (корень) + -ī-tāj- (суффикс) + -s (окончание)
  • atgriezt → at- (приставка) + griez- (корень) + -t (окончание)

Найдите корень в словах:

  • valde, valdīt, pārvaldīt, valdnieks → vald-
  • rakstīt, rakstnieks, uzrakstīt, parakstīt → rakst-
  • strādāt, strādnieks, nostrādāt, pārstrādāt → strād-
  • mācīt, mācītājs, izmācīt, pamācīt → māc-
  • runāt, runātājs, izrunāt, parunāt → run-
  • dziedāt, dziedātājs, izdziedāt, padziedāt → dzied-

Определите тип слова (первоначальное или производное):

  • saule → первоначальное (корень + окончание)
  • pasaule → производное (приставка + корень + окончание)
  • strādnieks → производное (корень + суффикс + окончание)
  • sniegs → первоначальное (корень + окончание)
  • pārvaldnieks → производное (приставка + корень + суффикс + окончание)
  • kliegšana → производное (корень + суффикс + окончание)
