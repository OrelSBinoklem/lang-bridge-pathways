(Простые времена страдательного залога)

💡 Что такое простые времена страдательного залога в латышском языке?
Страдательный залог образуется с помощью глагола "tikt" (получаться, становиться) в разных временах и причастия страдательного залога прошедшего времени. Объект действия становится подлежащим предложения.

📝 Образование страдательного залога

Структура страдательного залога:

Страдательный залог образуется по формуле:

  • Глагол "tikt" + причастие страдательного залога прошедшего времени
  • Причастие согласуется с подлежащим по роду и числу
  • Мужской род: -ts (ед.ч.), -ti (мн.ч.)
  • Женский род: -ta (ед.ч.), -tas (мн.ч.)

Примеры образования:

  • aicinā-t (приглашать) → aicinā-ts/aicinā-ta (приглашённый/приглашённая)
  • rakstī-t (писать) → rakstī-ts/rakstī-ta (написанный/написанная)
  • lasī-t (читать) → lasī-ts/lasī-ta (прочитанный/прочитанная)
  • būvē-t (строить) → būvē-ts/būvē-ta (построенный/построенная)

⚠️ Важные особенности

🔍 Особенности простых времен страдательного залога:
  • Страдательный залог образуется с помощью глагола "tikt"
  • Глагол "tikt" спрягается по временам: tieku/tiec/tiek (наст.), tiku/tiki/tika (прош.), tikšu/tiksi/tiks (буд.)
  • Причастия страдательного залога согласуются с подлежащим по роду и числу
  • Мужской род: -ts/-ti, женский род: -ta/-tas
  • Страдательный залог показывает, что действие испытывается подлежащим
  • Цветовое выделение помогает различать мужской и женский род
  • Объект действия становится подлежащим предложения
  • Исполнитель действия может быть выражен творительным падежом
💡 Практический совет: Запомните спряжение глагола "tikt" и формы причастий страдательного залога. Обратите внимание на согласование причастий с родом и числом подлежащего. Страдательный залог используется, когда важнее объект действия, чем его исполнитель.

  • es (настоящее) → tieku
  • tu (прошедшее) → tiki

  • mēs (настоящее) → tiekam
  • jūs (прошедшее) → tikāt
  • viņi (будущее) → tiks

Образуйте причастия страдательного залога:

  • aicināt (жен.р., ед.ч.) → aicināta
  • aicināt (муж.р., мн.ч.) → aicināti
  • aicināt (жен.р., мн.ч.) → aicinātas
  • rakstīt (муж.р., ед.ч.) → rakstīts
  • lasīt (жен.р., ед.ч.) → lasīta
